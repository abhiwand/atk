Detailed Description

Adds all of the edges and vertices into the their respective hashmaps. The

edgePropertyTable

hash is keyed by edge id (src, dst, label) with the value being the property map. The

vertexNameToTitanID

is keyed by vertex id, which should be a

StringType

or

LongType

with the value being the titan ID for the vertex.
