The symlinks are created to allow IntelAnalytics softare to reference
the cluster configurations files via /etc, instead of having to change
the IntelAnalytics RPMs directly:

yzou@zybuntu:for n in pig hadoop hbase titan titan-server; do ls -l etc/${n}; done
