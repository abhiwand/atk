Detailed Description

Remove any duplicate edges and vertices. If duplicates are found, either merge their property maps or call an optional Edge/Vertex reducer.

This will be called on the property graph element as we are iterating through the list received by the reducer.

All arguments are extracted from the argument builder and are all mandatory, except the

edgeReducerFunction

and the

vertexReducerFunction

.

  • edgeSet
    Hashtable with the current list of merged edges.
  • vertexSet
    Hashtable with the current list of merged vertices.
  • edgeReducerFunction
    Optional edge reducer function.
  • vertexReducerFunction
    Optional vertex reducer function.
  • vertexLabelMap
    List of vertex labels to be used for writing rdf output.
  • noBiDir
    Are we cleaning bidirectional edges. If true then remove bidirectional edge.
