Detailed Description

PropertyGraphElementTuple is the tuple type processed by the GB 2.0 (alpha) dataflow.

PropertyGraphElementTuple contains a list of property graph elements. Currently, GB UDFs know how to process tuples of type PropertyGraphElementTuple.
