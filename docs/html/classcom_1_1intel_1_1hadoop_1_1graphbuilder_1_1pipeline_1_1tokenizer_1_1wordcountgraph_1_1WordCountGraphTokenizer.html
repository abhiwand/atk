Detailed Description

GraphTokenizer

class that converts a Wiki page (presented as into a set of vertices and edges by the following rules:

  • There is a vertex for each wiki page.
  • There is a vertex for each word.
  • There is a "contains" edge between every page and every word that it contains.
  • The "contains" edge between a page and a word contains the frequency of the word in the page at the property "wordCount"
