Detailed Description

RegexExtractAllMatches returns a bag of all matched strings given a string and a regular expression.
This implementation is based on Pig's built in REGEX_EXTRACT_ALL UDF. The first element of the given tuple is the source string and the second element is the regular expression. RegexExtractAllMatches UDF only captures the first group specified in the regular expression.
